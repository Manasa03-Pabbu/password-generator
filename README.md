# 🔐 Smart Password Generator

A modern and secure password generator built using HTML, CSS, and JavaScript. This application creates strong passwords based on user preferences while providing real-time strength analysis and an intuitive user interface.

## 🚀 Features

* Generate secure and random passwords
* Customizable password length
* Password strength indicator
* Show/Hide password functionality
* Copy password to clipboard
* Dark/Light mode toggle
* Recent password history using Local Storage
* Responsive design for mobile and desktop
* Modern cybersecurity-inspired UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎯 How It Works

1. Enter your name, favorite color, and favorite animal.
2. Select the desired password length.
3. Click **Generate Password**.
4. View password strength instantly.
5. Copy the generated password with one click.
6. Access recently generated passwords from history.
